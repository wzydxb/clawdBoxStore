# 固定资产管理

## 适用场景
- 新资产采购入库登记
- 年度/季度资产盘点
- 资产调拨、报废、处置
- 折旧计算和资产报表

---

## 资产全生命周期（5 阶段）

```
规划采购 → 部署使用 → 维护保养 → 折旧核算 → 报废处置
   ↓          ↓          ↓          ↓          ↓
 预算审批    标签登记    预防性维护   月度计提    残值回收
 资本化判断  责任人分配   故障维修    减值测试    核销出账
```

## 资产台账（固定资产登记簿 FAR）

> FAR（Fixed Asset Register）是资产管理的单一事实来源。每项资产必须在 FAR 中有且仅有一条记录。

**台账字段：**

| 字段 | 说明 |
|------|------|
| 资产编号 | 格式：类别码-年份-序号（如 IT-2026-001） |
| 资产名称 | 具体名称+品牌型号 |
| 类别 | IT设备/办公家具/交通工具/其他 |
| 购入日期 | YYYY-MM-DD |
| 购入金额 | 含税金额（元）|
| 供应商 | 采购供应商 |
| 使用部门 | 当前使用部门 |
| 责任人 | 当前保管人（Custodian） |
| 存放位置 | 办公区域/工位编号 |
| 状态 | 在用/闲置/维修中/已报废 |
| 使用年限 | 年 |
| 折旧方法 | 直线法/加速法 |
| 标签/条码 | 条形码或 RFID 标签编号 |

**最佳实践（CPCON 标准）：**
- 所有资产必须贴条形码/RFID 标签
- 高价值资产（>1万元）附照片存档
- 盘点至少双人执行
- FAR 与总账（GL）定期核对

## 折旧方法对比

| 方法 | 特点 | 适用场景 |
|------|------|----------|
| 直线法 | 每年折旧额相等 | 默认方法，适合大多数资产 |
| 双倍余额递减法 | 前期折旧多后期少 | IT设备、技术更新快的资产 |
| 年数总和法 | 加速折旧 | 早期产出高的资产 |
| 工作量法 | 按使用量计提 | 车辆（按里程）、生产设备（按产量） |

**直线法公式：**
```
年折旧额 = (原值 - 残值) ÷ 使用年限
残值 = 原值 × 残值率（通常 5%）
```

**常见使用年限参考（IAS 16 / 企业会计准则）：**

| 资产类别 | 使用年限 | 残值率 | 推荐折旧方法 |
|----------|----------|--------|-------------|
| 电脑/笔记本 | 3年 | 5% | 双倍余额递减 |
| 服务器/网络设备 | 5年 | 5% | 直线法 |
| 办公家具 | 5年 | 5% | 直线法 |
| 车辆 | 4年 | 5% | 工作量法 |
| 空调/电器 | 5年 | 5% | 直线法 |

## 盘点流程（5 阶段）

> 参考 CPCON 资产盘点最佳实践。

### 阶段一：盘点准备
- 确定盘点范围（全面/抽样）
- 组建盘点小组（最少双人）
- 导出当前 FAR 生成盘点表
- 准备扫描设备/手持终端
- 通知各部门配合

### 阶段二：实物核查
- 逐项扫描条码/RFID
- 核对：编号、名称、位置、状态、责任人
- 高价值资产拍照留证
- 找不到的资产执行「二次搜索」协议

### 阶段三：账实核对
- FAR 记录 vs 实物盘点结果
- 标记差异：盘盈（有实物无账）/ 盘亏（有账无实物）/ 位置异动 / 状态变化
- 调查差异原因

### 阶段四：报告输出

| 指标 | 数值 |
|------|------|
| 资产总数 | X 项 |
| 盘点覆盖率 | X% |
| 账实相符率 | X%（目标 > 98%） |
| 盘盈 | X 项（金额 X 元） |
| 盘亏 | X 项（金额 X 元） |
| 需处置 | X 项 |

### 阶段五：整改落实
- 更新 FAR 台账
- 盘亏资产追责
- 盘盈资产补登
- 优化内控措施
- 确定下次盘点时间

## 报废处置

**报废条件（满足任一）：**
- 超过使用年限且无维修价值
- 维修成本 > 残值的 50%
- 技术淘汰无法满足工作需求
- 安全隐患无法消除

**处置方式：**
- 内部调拨（转给其他部门）
- 出售回收（二手市场/废品回收）
- 捐赠（公益机构，可抵税）
- 报废销毁（无残值资产）

**流程：**使用部门申请 → 技术鉴定 → 财务确认账面值 → 审批 → 执行处置 → 台账核销

## 典型陷阱

- **Ghost Assets（幽灵资产）**：账上有但实物已丢失/报废，虚增资产总额
- **Zombie Assets（僵尸资产）**：实物在但账上没登记，盘点永远对不上
- **人员离职不交接资产**：应纳入离职清单必办项
- **只盘数量不盘状态**：设备坏了/闲置但还挂「在用」
- **折旧方法与实际脱节**：IT设备用直线法 5 年折旧，实际 3 年就淘汰
