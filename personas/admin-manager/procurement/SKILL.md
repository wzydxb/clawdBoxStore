# 采购管理

## 适用场景
- 部门提交采购需求（办公用品、设备、服务）
- 供应商合同到期需要续签或更换
- 需要对现有供应商进行评估和优化
- 采购品类战略规划

---

## Kraljic 矩阵（采购品类分类）

> Peter Kraljic, 1983. 按「供应风险」×「利润影响」将采购品分为四类，每类采取不同策略。

```
              利润影响 高
                 │
    杠杆型        │       战略型
  竞争性招标       │     长期合作伙伴
  压低价格         │    联合开发/深度绑定
                 │
─────────────────┼─────────────────
                 │
    常规型        │       瓶颈型
  简化流程         │     保障供应
  自动化采购       │    寻找替代/安全库存
                 │
              利润影响 低
       供应风险 低          供应风险 高
```

**实操：**
- 列出所有采购品类，逐一标注供应风险和利润影响
- 归入四象限，按象限选策略
- 每季度复审一次（市场变化可能导致品类迁移）

## A.T. Kearney 七步战略采购

| 步骤 | 动作 | 输出 |
|------|------|------|
| 1. 品类画像 | 分析历史支出、供应商、合同 | 品类 Fact Pack |
| 2. 供应市场分析 | 市场结构、竞争格局、趋势 | 市场分析报告 |
| 3. 制定策略 | 基于 Kraljic 选择采购策略 | 品类策略文档 |
| 4. 选择采购方式 | RFP/RFQ/电子竞价/直接谈判 | 采购方案 |
| 5. 谈判与选定 | 多轮谈判、条款敲定 | 中标通知 |
| 6. 实施与整合 | 签合同、系统录入、交接 | 合同 + SOP |
| 7. 跟踪与评估 | KPI 监控、年度复盘 | 供应商评分卡 |

## TCO 总成本分析

> 采购决策看 TCO（Total Cost of Ownership），不看单价。

**三层成本模型：**

| 层级 | 成本项 | 举例 |
|------|--------|------|
| 交易前 | 寻源、资质审查、谈判、法务 | 人工时间、差旅、律师费 |
| 交易中 | 单价、运费、关税、付款条件 | 采购金额本身 |
| 交易后 | 退换、维修、停机、培训、报废 | 售后、维保、处置费 |

**比价时必须计算：**
```
TCO = 采购单价 + 运费 + 安装费 + 年维保费 × 使用年限 + 培训费 + 耗材费 - 残值
```

## 供应商评分卡

**加权评分模型：**

| 维度 | 权重 | 评分标准（10分制） |
|------|------|-------------------|
| 成本与TCO | 30% | 最低TCO=10分，按比例递减 |
| 质量 | 25% | 合格率100%=10分，95%+=8分，90%+=5分 |
| 交付 | 20% | 准时率100%=10分，延迟1天=8分，延迟3天+=5分 |
| 财务稳定性 | 10% | 信用评级、营收趋势、BCP |
| 技术能力 | 10% | 研发投入、技术路线图 |
| 合规/ESG | 5% | 环保认证、劳工合规 |

**评级输出：**
- ≥85分 → 战略供应商（优先合作）
- 70-84分 → 优选供应商（正常合作）
- 60-69分 → 合格供应商（有条件使用）
- <60分 → 淘汰/整改

**数据获取工具：**
- 企业背调 → `skill_view("data-acquisition/enterprise")`
- 商品比价 → `skill_view("data-acquisition/ecommerce")`

## 合同管理

**合同必审条款：**
- 价格条款（含税/不含税、是否含运费）
- 付款方式和账期（30/60/90天）
- 交货时间和验收标准
- 违约责任和赔偿条款
- 合同有效期和自动续签条件
- 保密条款和知识产权

**合同台账字段：**
合同编号 | 供应商 | 品类 | 金额 | 签署日期 | 到期日期 | 状态 | 负责人

**到期提醒：**
- 到期前 60 天：评估是否续签
- 到期前 30 天：启动续签或更换
- 到期前 7 天：确认新合同已签署

## 典型陷阱

- **只看单价不看TCO**：运费、安装费、维保费容易漏算，总成本可能翻倍
- **供应商过于集中**：单一供应商占比 > 60% 有断供风险（瓶颈型尤其危险）
- **合同到期才想起续签**：应提前 60 天启动流程
- **跳过市场分析直接比价**：不了解市场结构就谈判，容易被供应商牵着走
- **评分不跟踪**：签完合同不监控 KPI，供应商服务逐渐下滑
