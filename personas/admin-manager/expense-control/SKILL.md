# 行政费用管控

## 适用场景
- 年度/季度行政预算编制
- 月度预算执行跟踪和偏差分析
- 报销审核
- 行政成本优化

---

## 零基预算法（Zero-Based Budgeting）

> ZBB 不以上年预算为基础，每个费用科目从零开始论证其存在的必要性。适合行政费用管控，因为行政开支容易年年照抄、逐年膨胀。

**ZBB 六步流程：**

| 步骤 | 动作 | 输出 |
|------|------|------|
| 1. 定义目标 | 行政费用要支撑什么业务目标？ | 目标清单 |
| 2. 识别活动 | 将行政工作拆成独立活动单元 | 决策单元列表 |
| 3. 评估排序 | 每个活动按业务价值排序 | 优先级矩阵 |
| 4. 论证费用 | 每个活动分 3 档预算（最低/标准/理想） | 分级预算包 |
| 5. 分配资源 | 按排序从上往下分配，预算用完即止 | 最终预算 |
| 6. 监控复审 | 月度跟踪执行率，季度复审 | 偏差报告 |

**适用原则：**
- 固定费用（租金、物业）据实编，不适用 ZBB
- 变动费用（采购、差旅、招待）适合 ZBB
- 每年至少一个品类做 ZBB 深度审视

## 行政预算科目

| 一级科目 | 二级科目 | 预算依据 |
|----------|----------|----------|
| 办公费用 | 办公用品、打印耗材、文具 | 人均月耗 × 人数 |
| 租赁费用 | 办公租金、物业费、停车费 | 合同约定 |
| 设备费用 | IT设备、办公家具、维修 | 采购计划 + 维修预算 |
| 差旅费用 | 机票、酒店、交通、餐补 | 历史数据 × 出差计划 |
| 通讯费用 | 电话、网络、邮寄 | 历史均值 |
| 水电费用 | 水费、电费、空调费 | 历史均值 × 季节系数 |
| 车辆费用 | 油费、保险、维修、停车 | 车辆数 × 单车成本 |
| 招待费用 | 商务接待、会议茶歇 | 业务量预估 |
| 绿化保洁 | 保洁外包、绿植租摆 | 合同约定 |
| 其他 | 团建、节日福利、杂项 | 人数 × 标准 |

## 行业基准（G&A 占收入比）

> 数据来源：APQC（美国生产力与质量中心）、The Hackett Group

| 行业 | G&A 占收入比 | 说明 |
|------|-------------|------|
| 制造业 | 5-15% | 固定资产重，行政占比低 |
| 科技/互联网 | 15-25% | 人力密集，办公成本高 |
| 专业服务 | 20-30% | 人均办公成本高 |
| 零售/消费 | 8-15% | 门店分散 |

**关键效率指标：**

| 指标 | 计算方式 | 标杆值 |
|------|----------|--------|
| 行政成本/人 | 总行政费用 ÷ 员工人数 | 因城市而异 |
| 办公面积/人 | 总办公面积 ÷ 员工人数 | 8-12㎡ |
| 预算执行率 | 实际支出 ÷ 预算 × 100% | 90-105% |
| 采购节省率 | (预算-实际) ÷ 预算 × 100% | > 10% |

## 月度费用分析模板

```
【X月行政费用分析】

总预算：XX 万 | 已执行：XX 万（执行率 X%）| 剩余：XX 万

异常项（偏差 > 15%）：
· [科目]：预算X万 / 实际X万 / 超支X万
  原因：[具体原因]
  建议：[调整措施]

本月节省亮点：
· [措施]：节省 X 万（[说明]）

下月预测：
· 预计支出 X 万，[是否需要调整预算]
```

**偏差处理规则：**
- ≤ 10%：正常波动，不需处理
- 10%-20%：需说明原因
- > 20%：需审批调整预算或压缩后续支出

## 报销审核要点

| 检查项 | 要求 |
|--------|------|
| 发票真伪 | 扫码验真，税号正确 |
| 发票抬头 | 与公司名称完全一致 |
| 日期合理性 | 报销日期在发生日期后 30 天内 |
| 金额匹配 | 发票金额 = 报销金额 |
| 审批签字 | 部门负责人已签字 |
| 附件完整 | 出差有审批单，招待有客户名单 |
| 标准合规 | 差旅标准、招待标准未超标 |

## 成本优化（快速见效）

| 优化方向 | 具体措施 | 预期节省 |
|----------|----------|----------|
| 集中采购 | 合并同类需求批量采购 | 10-20% |
| 合同谈判 | 年度框架协议锁定价格 | 5-15% |
| 能源管理 | 空调定时、照明感应、节能设备 | 10-15% |
| 无纸化 | 电子审批、共享打印、双面打印 | 20-30% |
| 差旅管理 | 提前预订、协议酒店、视频会议替代 | 15-25% |
| 供应商整合 | 减少供应商数量，提高议价力 | 5-10% |

## 典型陷阱

- **只控大头不管小项**：小额高频费用（快递、打印、文具）积少成多
- **预算编完不跟踪**：年底才发现超支，已无法挽回
- **一刀切砍预算**：应区分必要支出和弹性支出，砍错了影响业务
- **Kraft Heinz 教训**：过度 ZBB 砍了品牌和研发投入，长期竞争力下降。ZBB 应该是战略性重新分配资源，不是简单削减
