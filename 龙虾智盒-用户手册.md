# 龙虾智盒 · 用户操作手册

> 你的 AI 数字分身已就绪——这份手册告诉你怎么快速上手，让它真正帮你干活。

---

## 从其他龙虾换过来？直接继承，不用重新养

如果你之前用过 openclaw、easyclaw、qclaw、autoclaw 或其他 AI Bot，**不需要从头开始**。

加完 Bot 后，第一句话告诉它：

```
我之前用的是[旧Bot名字]，用了[多长时间]
```

它会引导你把旧 Bot 养出来的习惯、偏好、进行中的任务都迁移过来——不用重新说你的背景，不用重新设置风格，直接接着用。

> 有旧的聊天记录截图或文字？粘几段给它，它能直接从里面学到你的习惯。

---

## 第一步：扫码加 Bot

盒子背面贴有专属微信二维码，或联系交付人员获取。

添加后，**直接发一条消息**，Bot 会自动启动初始化引导。

> 💡 后续所有对话都在这个微信窗口里完成，不需要打开电脑或后台。

---

## 第二步：完成初始化（约3分钟）

首次发消息，Bot 会依次问你几个问题：

| 问题 | 示例回答 |
|------|----------|
| 你是做什么的？ | 我是产品经理 |
| 所在行业？ | 互联网/电商/医疗/制造业…… |
| 公司/产品名？ | XX科技，做企业SaaS |
| 选择思维原型 | 会列出16+大佬原型供选择 |

完成后，你的专属数字分身正式激活。**之后每次对话，它都记得你是谁、做什么的。**

## 日常使用：直接说，不用想格式

不需要背指令，用正常说话的方式就行。下面是最常用的场景：

---

```

```
```

```

## 常见问题

**Q：发消息没反应怎么办？**

等3-5秒，Bot 在思考中。如果超过30秒没回，发「/new」重开会话。

**Q：怎么开始一个新话题？**

直接说新话题就行，或发「/new」明确开新会话。

**Q：Bot 记不住上次说的内容？**

跨会话引用时加一句「上次我们聊的[话题]……」，Bot 会自动检索历史记忆。

**Q：怎么让别人也能加这个 Bot？**

```
分享一下，帮我生成二维码
```

**Q：怎么知道 Bot 会什么？**

直接问「你会什么」或「你能帮我做什么」，它会根据你的角色列出当前可用能力。

## 附录：挂载盒子网盘

盒子内置 SMB 网盘，把文件丢进去就能被 AI 读取。

**Mac：**
1. Finder → 前往 → 连接服务器
2. 输入 `smb://[盒子IP地址]`
3. 用户名/密码由交付人员提供

**Windows：**
1. 打开文件资源管理器
2. 地址栏输入 `\\[盒子IP地址]`
3. 输入凭据

**手机（需同一WiFi）：**
推荐用「ES文件浏览器」或「nPlayer」连接 SMB。

> 💡 盒子 IP 地址印在设备底部标签上，或联系交付人员确认。
