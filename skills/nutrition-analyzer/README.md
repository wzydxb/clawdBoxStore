# nutrition-analyzer

## 简介

营养分析与建议 Skill。分析营养数据、识别营养模式、评估营养状况，并根据个人情况提供个性化营养建议。

## 主要功能

- 饮食记录分析：解析用户描述的饮食内容，计算热量和营养素摄入
- 营养素评估：评估蛋白质、碳水、脂肪、维生素、矿物质等摄入是否达标
- 营养模式识别：发现饮食结构中的规律性问题（如长期缺铁、蛋白质不足等）
- 个性化建议：结合年龄、性别、活动量、健康目标给出针对性改善建议
- 食物营养查询：查询特定食物的营养成分

## 使用方式

通过 OpenClaw 对话界面直接描述饮食情况，例如：

```
我今天吃了：早餐燕麦粥一碗，午餐米饭加红烧肉，晚餐沙拉。帮我分析一下营养情况。
```

```
我最近总是感觉疲劳，饮食上可能缺什么？
```

```
查一下鸡胸肉的营养成分
```

也可以上传饮食记录文件（CSV/JSON 格式）进行批量分析。

## 依赖 / 前置条件

- 无需外部 API Key
- Python 3.8+
- 依赖见 `requirements.txt`

## 注意事项

- 营养分析基于通用食物数据库，实际营养含量因食材产地、烹饪方式有所差异
- 本工具提供的建议仅供参考，不构成医疗或临床营养建议
- 有特殊疾病（糖尿病、肾病等）的用户，饮食调整前请咨询注册营养师或医生
