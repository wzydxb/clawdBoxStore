## 使用说明

1. **快速定位**: 根据需求类型，先确定大类（1-9），再细化到子类
2. **关键词搜索**: 在对应分类下搜索具体功能关键词
3. **模型选择**: 根据性能、成本、语言支持等因素选择合适的模型
4. **文档查阅**: 每个 API 都有详细的 MD 文档，包含参数说明和示例代码
