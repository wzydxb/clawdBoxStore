# short-video-script-v1-0-0

## 简介

短视频脚本创作助手，针对抖音、快手、B站、视频号等主流平台，生成高点击率爆款标题和完整视频脚本。

## 主要功能

- 生成符合各平台调性的爆款标题
- 输出完整短视频脚本（开场钩子、正文节奏、结尾引导）
- 支持多种内容类型：知识科普、产品种草、情感共鸣、搞笑娱乐等
- 针对不同平台风格差异化输出

## 使用方式

通过 OpenClaw 对话触发，示例：

```
帮我写一个抖音视频脚本，主题是"上班族如何在家做副业"，时长 60 秒
帮我生成 5 个 B站标题，内容是 Python 入门教程
```

支持指定：
- 目标平台（抖音 / 快手 / B站 / 视频号）
- 视频时长（15s / 30s / 60s / 3min+）
- 内容方向和目标受众

## 依赖 / 前置条件

- 无需额外 API Key
- 依赖 OpenClaw 内置 LLM 能力

## 注意事项

- 生成内容仅供参考，建议结合实际账号风格调整
- 爆款标题有一定套路性，使用时注意与账号定位匹配
- 平台规则持续变化，敏感词和违禁内容需自行把关
