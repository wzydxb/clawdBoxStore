# legal-counsel-v2

## 简介

资深法律顾问 Skill，拥有 20+ 年执业经验的虚拟法律专家。提供法律咨询、合同起草、合规审查等专业服务，覆盖民商事、劳动、知识产权等常见法律领域。

## 主要功能

- 法律问题解答：就合同纠纷、劳动争议、侵权责任等常见问题给出专业意见
- 合同起草：根据需求起草买卖合同、劳动合同、保密协议、合作协议等
- 合同审查：逐条审核合同条款，识别风险点并提出修改建议
- 合规检查：针对特定业务场景进行合规性评估
- 法律文书模板：提供常用法律文书的标准模板

## 使用方式

通过 OpenClaw 对话界面直接描述法律需求，例如：

```
帮我审查这份劳动合同，重点看竞业限制和保密条款。
```

```
我需要起草一份软件开发服务合同，甲方是我们公司，乙方是外包团队。
```

```
我的员工单方面离职，我能要求赔偿违约金吗？
```

## 依赖 / 前置条件

- 无需外部 API Key
- 无需额外安装依赖

## 注意事项

- 本 Skill 提供的法律意见仅供参考，不构成正式法律服务
- 涉及重大诉讼、仲裁或高价值合同，建议咨询持牌律师进行确认
- 默认适用中国大陆法律体系，如需其他司法管辖区请在提问时明确说明
- 法律法规存在更新，使用前建议核实相关条文的现行有效版本
