# 张雪峰.skill

## 简介

基于张雪峰的著作、采访、公开语录等资料深度调研提炼，复现其思维框架与表达方式，帮助用户以张雪峰视角分析问题、给出建议。

## 主要功能

- 以张雪峰的语言风格和思维逻辑回答问题
- 提供志愿填报、专业选择、职业规划等领域的建议
- 复现其标志性的"接地气"表达方式与说服逻辑
- 基于其核心心智模型分析教育、就业、社会现象

## 使用方式

直接向 skill 提问，例如：

- "我高考 550 分，想学计算机，应该怎么选学校？"
- "文科生有哪些好就业的专业？"
- "我孩子想学艺术，你怎么看？"

skill 会以张雪峰的风格和逻辑给出回答。

### 参考资料

skill 基于以下研究文件构建（位于 `references/research/`）：

| 文件 | 内容 |
|------|------|
| `writings` | 著作与文章摘录 |
| `conversations` | 访谈与对话记录 |
| `expression-dna` | 表达风格与语言特征 |
| `external-views` | 外部评价与争议观点 |
| `decisions` | 典型决策案例与逻辑 |
| `timeline` | 人生经历与成长脉络 |

## 依赖 / 前置条件

- 无需额外 API key

## 注意事项

- 本 skill 为风格模拟，仅供参考，不构成正式志愿填报建议
- 实际填报请结合当年分数线、个人情况综合判断
- skill 内容基于公开资料，不代表张雪峰本人实时观点
