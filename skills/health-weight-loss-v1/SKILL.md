# 健康减肥助手 v1.0.0

基于运动科学和营养学原理，为你生成个性化的减重方案，包括运动计划和饮食指导。

---

## 🤖 AI 对话引导流程

**当用户表达减肥/减重/健身需求时，按以下步骤执行：**

### Step 1：确认用户意图

用户说"我想减肥"、"帮我制定减重计划"等→ 确认具体需求：

```
我可以帮你制定个性化的健康减重方案，包括：
- 运动计划（根据你的时间和条件）
- 饮食指导（科学搭配，不节食）
- 进度追踪建议

请问你希望重点关注哪方面？还是全套方案？
```

### Step 2：收集用户信息（分批询问）

**每次最多问 4-5 个问题**，按以下维度收集：

#### 基础信息（必填）
| 字段 | 说明 | 示例 |
|------|------|------|
| gender | 性别 | 男/女 |
| age | 年龄 | 28 |
| height_cm | 身高（厘米） | 170 |
| current_weight_kg | 当前体重（公斤） | 75 |
| target_weight_kg | 目标体重（公斤） | 65 |

#### 身体状况（必填）
| 字段 | 说明 | 示例 |
|------|------|------|
| health_conditions | 健康状况/既往病史 | 无/高血压/糖尿病/膝盖损伤 |
| physical_limitations | 身体限制 | 膝盖不好/腰椎问题/无 |
| current_fitness_level | 当前运动基础 | 零基础/偶尔运动/经常运动 |

#### 时间安排（必填）
| 字段 | 说明 | 示例 |
|------|------|------|
| weekly_workout_days | 每周可运动天数 | 3-5天 |
| daily_workout_minutes | 每次运动时长（分钟） | 30-60 |
| preferred_workout_time | 偏好运动时段 | 早晨/午休/晚上 |

#### 条件与偏好（选填）
| 字段 | 说明 | 示例 |
|------|------|------|
| available_equipment | 可用器械 | 无/哑铃/健身房/瑜伽垫 |
| preferred_exercise_types | 喜欢的运动类型 | 跑步/游泳/瑜伽/力量训练 |
| target_body_parts | 重点塑形部位 | 腹部/腿部/全身 |
| dietary_restrictions | 饮食限制 | 素食/清真/过敏食物 |
| cooking_condition | 做饭条件 | 自己做饭/外卖/公司食堂 |

### Step 3：计算关键指标

收集完基础信息后，计算并展示：

```
根据你的信息，我为你计算了以下指标：

📊 **身体数据分析**
- BMI: [计算结果] ([偏瘦/正常/超重/肥胖])
- 理想体重范围: [XX-XX] kg
- 需减重: [X] kg
- 建议减重周期: [X] 个月（健康速度：每周0.5-1kg）
- 每日建议热量摄入: [XXXX] kcal
- 基础代谢率(BMR): [XXXX] kcal
```

**计算公式**：
- BMI = 体重(kg) / 身高(m)²
- BMR（男性）= 88.362 + (13.397 × 体重kg) + (4.799 × 身高cm) - (5.677 × 年龄)
- BMR（女性）= 447.593 + (9.247 × 体重kg) + (3.098 × 身高cm) - (4.330 × 年龄)
- TDEE = BMR × 活动系数（久坐1.2/轻度1.375/中度1.55/高强度1.725）
- 减脂热量 = TDEE - 500（制造每日500kcal热量缺口）

### Step 4：生成个性化方案

确认所有信息后，生成完整方案：

```
好的，我将为你生成包含以下内容的全套方案：
1. 运动计划（4周渐进式）
2. 饮食指导（每日食谱框架）
3. 执行建议与注意事项

确认生成吗？
```

生成时调用 Python 脚本：

```bash
cd ~/.openclaw/skills/health-weight-loss-v1
python scripts/generate_plan.py \
  --gender 男 \
  --age 28 \
  --height 170 \
  --current-weight 75 \
  --target-weight 65 \
  --fitness-level 零基础 \
  --workout-days 4 \
  --workout-minutes 45 \
  --equipment 无 \
  --output ~/.openclaw/workspace/health_plans/user_plan_20240324.md
```

### Step 5：展示并解释方案

生成完成后，向用户展示：

## 🏃 运动方案设计原则

### 方案结构

每个4周计划包含：

| 周次 | 重点 | 运动类型配比 |
|------|------|-------------|
| 第1周 | 适应期 | 低强度有氧70% + 基础力量30% |
| 第2周 | 提升期 | 中等有氧60% + 力量训练40% |
| 第3周 | 强化期 | 间歇训练50% + 力量训练50% |
| 第4周 | 巩固期 | 综合训练，测试体能进步 |

### 运动类型库

根据可用器械条件自动匹配：

#### 无器械/居家
- 有氧：开合跳、高抬腿、波比跳、登山跑
- 力量：俯卧撑、深蹲、弓步蹲、平板支撑、仰卧起坐
- 拉伸：瑜伽基础动作、静态拉伸

#### 哑铃/简单器械
- 增加：哑铃划船、哑铃推举、哑铃深蹲、硬拉

#### 健身房
- 增加：杠铃训练、器械训练、游泳、动感单车

### 强度分级

| 级别 | 心率区间 | 主观感受 | 适用人群 |
|------|---------|---------|---------|
| 轻度 | 最大心率的50-60% | 可以轻松说话 | 零基础/大体重 |
| 中等 | 最大心率的60-70% | 呼吸加快，勉强说话 | 有基础者 |
| 高强度 | 最大心率的70-85% | 呼吸急促，难以说话 | 进阶训练 |

> 最大心率 = 220 - 年龄

## 🥗 饮食指导原则

### 营养配比

| 营养素 | 占比 | 作用 |
|--------|------|------|
| 蛋白质 | 25-30% | 维持肌肉，增加饱腹感 |
| 碳水化合物 | 40-45% | 提供能量，优选复合碳水 |
| 脂肪 | 25-30% | 激素合成，优选不饱和脂肪 |
| 膳食纤维 | 25-30g/天 | 促进消化，增加饱腹感 |

### 饮食模式

**三餐分配**：
- 早餐：30%（高蛋白开启代谢）
- 午餐：40%（营养均衡，能量充足）
- 晚餐：25%（清淡为主，控制碳水）
- 加餐：5%（水果/坚果，防暴食）

**食物选择原则**：
- ✅ 优先：瘦肉、鱼、蛋、蔬菜、全谷物、豆类
- ⚠️ 适量：水果、坚果、乳制品
- ❌ 限制：精制糖、油炸食品、含糖饮料、加工食品

### 每日食谱框架示例

```
【早餐 约500kcal】
- 蛋白质：鸡蛋2个 / 牛奶250ml
- 碳水：全麦面包2片 / 燕麦50g
- 蔬果：番茄/黄瓜/苹果

【午餐 约700kcal】
- 蛋白质：鸡胸肉/鱼/瘦牛肉 150g
- 碳水：糙米/红薯/全麦面 150g
- 蔬菜：绿叶蔬菜 200g

【晚餐 约450kcal】
- 蛋白质：豆腐/鱼虾 100g
- 碳水：少量杂粮 100g
- 蔬菜：蔬菜汤/凉拌菜 200g

【加餐 约150kcal】
- 下午：坚果10g / 水果1个
- 运动前（如需要）：香蕉半根
```

## ⚠️ 安全与健康提醒

### 必须遵守的原则

1. **循序渐进**：不要突然大幅增加运动量或大幅减少热量
2. **倾听身体**：疼痛时停止，区分肌肉酸痛和受伤疼痛
3. **充足睡眠**：每天7-8小时，睡眠不足影响代谢
4. **保持水分**：每天饮水2000-2500ml，运动时额外补充
5. **定期评估**：每周称重，每两周测量围度，不要每天称重

### 以下情况请咨询医生

- BMI ≥ 28 或存在心血管疾病
- 有糖尿病、高血压等慢性病
- 关节或骨骼问题
- 孕期/哺乳期
- 长期未运动且年龄>40岁

### 危险信号（立即停止运动）

- 胸痛或胸闷
- 头晕或昏厥感
- 呼吸困难（非正常运动喘息）
- 心悸或心律不齐
- 恶心伴随冷汗

## 📊 进度追踪建议

### 记录指标

| 指标 | 频率 | 记录方式 |
|------|------|---------|
| 体重 | 每周1次（固定时间） | 早晨空腹 |
| 腰围 | 每2周1次 | 肚脐水平 |
| 体脂率 | 每月1次 | 体脂秤/健身房测量 |
| 运动完成度 | 每次 | 打卡记录 |
| 饮食记录 | 前2周建议每天 | 拍照/APP记录 |

### 调整时机

- **2周后无变化**：检查饮食记录，确认热量缺口
- **体重下降过快**（>1kg/周）：适当增加热量摄入
- **平台期**（>2周不变）：调整运动模式或饮食结构
- **感到过度疲劳**：增加休息日，降低强度

## ✅ 最佳实践

### 应该做的

- ✅ 设定合理目标（每周减0.5-1kg）
- ✅ 优先通过饮食创造热量缺口
- ✅ 力量训练+有氧结合，保留肌肉
- ✅ 保证蛋白质摄入（1.2-1.6g/kg体重）
- ✅ 找到可持续的生活方式，而非短期节食
- ✅ 允许偶尔的"欺骗餐"，保持心理平衡

### 不应该做的

- ❌ 极端节食（<1200kcal/天）
- ❌ 完全不吃碳水
- ❌ 每天称重导致焦虑
- ❌ 只做有氧忽视力量训练
- ❌ 追求快速减重（>1kg/周）
- ❌ 使用减肥药/代餐作为主要手段
