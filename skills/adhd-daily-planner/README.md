# ADHD 每日规划助手

## 简介

专为 ADHD 大脑设计的规划系统，提供时间盲友好的任务规划、执行功能支持和日常结构搭建。核心理念是现实时间估算、多巴胺感知任务设计，帮助神经多样性思维建立真正可执行的日常系统。

## 主要功能

- 时间盲友好的任务分解：将大任务拆成具体可感知的小步骤，附带现实时间估算
- 多巴胺菜单设计：根据当前精力状态匹配合适的任务，避免强迫执行
- 执行功能支持：提供启动提示、任务切换缓冲、完成确认等辅助结构
- 日常结构模板：早晨启动、工作时段、收尾仪式等可复用的日程框架
- 卡壳应对策略：识别常见执行障碍并给出即时可用的绕过方案

## 使用方式

直接描述你今天需要完成的事情，或者告诉助手你目前的状态（精力高/低、焦虑/平静），助手会根据你的状态给出适合的规划建议。

示例提示：
- "我今天有三件事要做，但完全不知道从哪里开始"
- "我现在精力很低，帮我找一件能启动的小事"
- "帮我设计一个工作日的早晨启动流程"

## 依赖/前置条件

无需 API Key，无需额外配置。

参考文件（skill 内置，无需手动操作）：
- `references/dopamine-menu.md`：多巴胺菜单设计方法论
- `references/executive-function-toolkit.md`：执行功能支持工具包

## 注意事项

- 本 skill 提供规划建议，不替代专业 ADHD 诊断或治疗
- 建议结合实际情况灵活调整，不必强迫自己完全按计划执行
- 如果某个策略不适合你，直接告诉助手，它会换一种方式
