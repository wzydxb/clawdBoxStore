# 情感专家

## 简介

约会场景的消息教练，提供聊天回复建议、消息改写、DM 开场白设计和对话分析，帮助用户在约会沟通中更自然、更有吸引力。

## 主要功能

- 回复建议：分析对方消息，给出多个风格不同的回复选项
- 消息改写：将平淡或尴尬的消息改写得更有趣、更有吸引力
- DM 开场白：根据对方主页信息生成个性化的第一条消息
- 聊天分析：分析对话截图，判断对方的兴趣程度和情绪状态
- 约会邀约：帮助设计自然的约会邀请话术

## 使用方式

核心脚本：`scripts/wingman_store.py`，参考模板位于 `references/` 目录。

直接将聊天截图或消息内容发给助手，说明你的需求：

示例提示：
- "她发了这条消息，我应该怎么回？"
- "帮我把这条消息改得更有趣一点"
- "我想给这个人发第一条 DM，她的简介是..."
- "分析一下这段对话，她对我感兴趣吗？"

## 依赖/前置条件

- 无需 API Key
- 支持上传聊天截图（图片分析功能需要模型支持多模态输入）

## 注意事项

- 本 Skill 提供沟通建议，最终如何表达由你自己决定
- 建议保持真实，过度依赖脚本可能让对话显得不自然
- 不支持生成欺骗性或操控性内容
