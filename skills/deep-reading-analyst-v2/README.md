# 论文分析 / Deep Reading Analyst

深度阅读分析技能，运用10+种思维框架（SCQA、5W2H、批判性思维、逆向思维、心智模型、第一性原理、系统思维、六顶帽子等），将文章、论文、长篇内容转化为可落地的深度洞见。

## 功能特点

- **多源输入**：支持 URL、PDF、DOCX、TXT/MD 文件及直接粘贴文本，自动提取内容
- **四级深度**：Quick（15分钟）/ Standard（30分钟）/ Deep（60分钟）/ Research（120分钟+）
- **10+思维框架**：SCQA结构分析、5W2H完整性检查、批判性思维、逆向思维、心智模型、第一性原理、系统思维、六顶帽子、跨源对比
- **五种输出模式**：问题解决 / 学习笔记 / 写作参考 / 决策分析 / 知识激活

## 适用场景

- 深度理解复杂论文或文章
- 识别论证逻辑漏洞
- 从阅读材料中提取可行动洞见
- 制作学习笔记或知识摘要
- 多来源内容对比分析
- 将知识转化为实际应用

## 使用方法

直接提供内容来源即可：
- 粘贴文章/论文全文
- 提供 URL 链接
- 提供文件路径（PDF/DOCX/TXT/MD）

然后告诉技能你的分析目的和期望深度。
