# patent-drafting-v2

## 简介

专业专利申请起草工具。支持 USPTO（美国）、CNIPA（中国）、EPO（欧洲）等多个司法管辖区的专利申请文件起草，覆盖发明专利、实用新型等类型。

## 主要功能

- 权利要求书起草：生成独立权利要求和从属权利要求，确保保护范围合理
- 说明书撰写：技术背景、发明内容、具体实施方式的完整撰写
- 摘要生成：符合各局格式要求的摘要
- 合规检查：对照 CNIPA/USPTO/EPO 审查指南检查常见缺陷
- 格式验证：验证文件格式是否符合目标局提交要求
- 参考案例匹配：从 `references/patterns/` 中匹配相似技术领域的成功案例

## 使用方式

通过 OpenClaw 对话界面描述发明内容，例如：

```
我发明了一种新型电池散热结构，请帮我起草中国专利申请文件。
```

```
帮我检查这份权利要求书是否有保护漏洞。
```

参考资料位于 `references/` 目录：
- `cnipa_guidelines/`：中国专利局审查指南
- `patterns/`：各技术领域成功案例模板
- `sharp_edges/`：常见驳回理由及应对策略
- `validations/`：格式验证规则

## 依赖 / 前置条件

- 无需外部 API Key
- Python 3.8+（格式验证脚本）

## 注意事项

- 本工具生成的文件为初稿，正式提交前建议由专利代理人审核
- 不同司法管辖区的要求差异较大，跨国申请需分别适配
- 专利申请有时效要求（优先权期限 12 个月），请注意提交时间节点
- 涉及核心技术的专利，建议在提交前进行专利检索，评估授权可能性
