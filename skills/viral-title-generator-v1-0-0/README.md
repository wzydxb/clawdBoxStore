# viral-title-generator-v1-0-0

## 简介

爆款标题生成工具，针对小红书、公众号、抖音、知乎、B站、微博等主流平台，生成高点击率标题，内置平台规则和标题公式参考库。

## 主要功能

- 支持 6 大平台：小红书、微信公众号、抖音、知乎、B站、微博
- 内置多种爆款标题公式（悬念式、数字式、对比式、痛点式等）
- 参考真实爆款案例库生成标题
- 一次生成多个候选标题供选择
- 针对各平台调性和用户习惯差异化输出

## 使用方式

通过 OpenClaw 对话触发，示例：

```
帮我给小红书写 5 个标题，内容是"职场新人如何快速升职"
给这篇公众号文章生成标题：[粘贴文章摘要]
为抖音视频生成爆款标题，主题是家庭收纳技巧
```

核心脚本：
- `scripts/generate_titles.py`：标题生成主逻辑
- `references/examples/`：各平台爆款标题案例库
- `references/platform-rules/`：各平台标题规则和限制说明
- `references/title-formulas/`：标题公式模板库

## 依赖 / 前置条件

- Python 3.8+
- 依赖 OpenClaw 内置 LLM 能力，无需额外 API Key

## 注意事项

- 生成标题仅供参考，建议结合账号风格和受众特点调整
- 各平台对标题字数有限制（如小红书建议 20 字以内），生成时已按规则控制
- 过度使用套路化标题可能影响账号权重，建议适度使用
- 平台算法持续变化，参考案例库需定期更新
