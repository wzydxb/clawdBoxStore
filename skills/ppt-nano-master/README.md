# ppt-nano-master

## 简介

根据用户提供的主题或内容，自动生成多风格 PPT 文件。支持 14 种视觉风格，适合演示、教学、汇报等场景。

## 主要功能

- 根据主题或大纲自动生成完整 PPT
- 支持 14 种风格：白板、立体、毛毡风、黑板报等
- 可生成配套插图（调用 generate_image.py）
- 输出标准 .pptx 格式文件

## 使用方式

通过 OpenClaw 对话触发，示例：

```
帮我做一份关于"人工智能在教育中的应用"的 PPT，用黑板报风格，8 页
```

核心脚本：
- `scripts/build_pptx.py`：负责 PPT 结构生成与样式渲染
- `scripts/generate_image.py`：负责幻灯片配图生成
- `styles/`：存放 14 种风格的模板资源

## 依赖 / 前置条件

- Python 3.8+
- `python-pptx`、`Pillow` 等依赖（见 requirements.txt）
- 如需 AI 生图，需配置图像生成服务端点

## 注意事项

- 生成图片较多时耗时较长，建议先用纯文字版本验证内容
- 风格资源文件在 `styles/` 目录，不要随意删改
- 输出文件默认保存在工作目录，可在调用时指定路径
