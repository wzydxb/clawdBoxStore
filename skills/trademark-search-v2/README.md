# trademark-search-v2

## 简介

商标查询工具，支持同时检索中国（CNIPA）、美国（USPTO）和国际（WIPO）三大商标数据库，查看商标注册状态和详细信息，辅助注册前的可行性评估。

## 主要功能

- 查询中国国家知识产权局（CNIPA）商标数据库
- 查询美国专利商标局（USPTO）商标数据库
- 查询世界知识产权组织（WIPO）国际商标数据库
- 检查商标名称是否已被注册、注册详情（申请人、类别、状态、有效期等）
- 近似商标检索（音近 / 形近 / 义近）
- 尼斯分类 45 类完整参考，准确定位检索类别

## 使用方式

通过 OpenClaw 对话触发，示例：

```
查一下"龙虾助手"这个商标在中国有没有被注册
检查 "ClawdBox" 在美国和国际商标库的注册情况
帮我查询第 42 类软件服务类别下，"星图"商标的中国注册状态
```

## 依赖 / 前置条件

- 无需额外 API Key
- 数据通过 web_search 访问各数据库公开信息（直连接口受限，通过搜索引擎间接获取）

## 注意事项

- 查询结果仅供参考，不构成法律意见
- 商标是否可注册涉及近似度、混淆可能性、类别等复杂判断，正式申请前建议咨询专业商标代理机构
- 部分境外数据库访问在国内网络下可能较慢，如遇超时请重试
- CNIPA 数据以中文检索为主，英文品牌名建议同时查询拼音和英文
