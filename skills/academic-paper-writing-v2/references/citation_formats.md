## APA格式（第7版）

### 适用领域
心理学、教育学、社会科学、商学

### 期刊论文

**文内引用**：
```
单作者：（张三, 2020）或 Zhang (2020)
双作者：（张三 & 李四, 2020）
三位及以上：（张三 等, 2020）或 Zhang et al. (2020)
```

**参考文献列表**：
```
中文期刊：
张三, 李四, & 王五. (2020). 论文标题. 期刊名称, 35(2), 45-67.

英文期刊：
Smith, J., & Johnson, M. (2020). Title of the article. Journal Name, 35(2), 45-67. https://doi.org/10.xxx/xxx
```

### 图书

**参考文献列表**：
```
中文图书：
李四. (2018). 书名（第二版）. 出版社.

英文图书：
Brown, A. (2018). Book title (2nd ed.). Publisher Name.
```

### 网络资源

```
作者. (发布日期或更新日期). 标题. 网站名称. https://www.example.com/article
```

## MLA格式（第9版）

### 适用领域
人文学科、语言学、文学研究

### 期刊论文

**文内引用**：
```
(张三 45) 或 (Smith 45)
多位作者：(张三, 李四 45)
```

**参考文献列表**：
```
中文期刊：
张三, 李四. "论文标题." 期刊名称, vol. 35, no. 2, 2020, pp. 45-67.

英文期刊：
Smith, John, and Mary Johnson. "Title of the Article." Journal Name, vol. 35, no. 2, 2020, pp. 45-67.
```

### 图书

```
中文图书：
李四. 书名. 出版社, 2018.

英文图书：
Brown, Anna. Book Title. Publisher Name, 2018.
```

## Chicago格式（第17版）

### 适用领域
历史学、艺术学、部分社会科学

### 两种引用方式

**注释-参考文献制（Notes-Bibliography）**：

脚注格式：
```
首次引用：
1. 张三, 论文标题, 期刊名称 35, no. 2 (2020): 45-67.

再次引用：
2. 张三, 论文标题, 50.
```

参考文献列表：
```
张三. "论文标题." 期刊名称 35, no. 2 (2020): 45-67.

Smith, John. "Title of the Article." Journal Name 35, no. 2 (2020): 45-67.
```

**作者-日期制（Author-Date）**：

文内引用：
```
(张三 2020, 45)
```

参考文献列表：
```
张三. 2020. "论文标题." 期刊名称 35 (2): 45-67.
```

## GB/T 7714-2015格式

### 适用领域
中国国家标准，广泛用于国内期刊和学位论文

### 期刊论文

**参考文献列表**：
```
[1] 张三, 李四, 王五. 论文标题[J]. 期刊名称, 2020, 35(2): 45-67.
[2] Smith J, Johnson M. Title of the article[J]. Journal Name, 2020, 35(2): 45-67.
```

### 图书

```
[1] 李四. 书名[M]. 第二版. 出版地: 出版社, 2018: 100-120.
```

### 学位论文

```
[1] 王五. 论文标题[D]. 城市: 学校名称, 2020.
```

### 会议论文

```
[1] 张三. 论文标题[C]//会议名称. 会议地点: 主办方, 2020: 45-50.
```

### 电子资源

```
[1] 作者. 标题[EB/OL]. (发布日期)[引用日期]. https://www.example.com/article.
```

### 文献类型标识

| 标识 | 文献类型 |
|-----|---------|
| M | 图书 |
| J | 期刊文章 |
| D | 学位论文 |
| C | 会议论文 |
| R | 报告 |
| P | 专利 |
| N | 报纸文章 |
| EB/OL | 网络资源 |

## 文献数据JSON格式规范

### 输入格式（citation_formatter.py）

```json
{
  "format": "apa",
  "references": [
    {
      "type": "journal",
      "authors": ["张三", "李四", "王五"],
      "title": "基于深度学习的图像分类研究",
      "journal": "计算机学报",
      "year": 2020,
      "volume": "43",
      "issue": "2",
      "pages": "234-250",
      "doi": "10.11897/SP.J.1016.2020.00234"
    },
    {
      "type": "book",
      "authors": ["周志华"],
      "title": "机器学习",
      "publisher": "清华大学出版社",
      "year": 2016,
      "edition": "第1版"
    },
    {
      "type": "conference",
      "authors": ["Smith J", "Johnson M"],
      "title": "Deep Learning for Natural Language Processing",
      "conference": "Proceedings of ACL 2020",
      "year": 2020,
      "pages": "1234-1245"
    }
  ]
}
```

### 字段说明

| 字段 | 必需 | 说明 |
|-----|------|------|
| type | 是 | 文献类型：journal/book/conference/thesis/web |
| authors | 是 | 作者列表，格式："姓 名"或"姓 名首字母" |
| title | 是 | 论文/图书标题 |
| year | 是 | 出版年份 |
| journal | 条件 | 期刊名称（type=journal时必需） |
| volume | 否 | 卷号 |
| issue | 否 | 期号 |
| pages | 条件 | 页码范围 |
| publisher | 条件 | 出版社（type=book时必需） |
| edition | 否 | 版次 |
| doi | 否 | DOI号 |
| url | 条件 | 网址（type=web时必需） |

### 输出格式

脚本输出包含两部分：

**1. formatted_references.txt**（参考文献列表）：
```
张三, 李四, & 王五. (2020). 基于深度学习的图像分类研究. 计算机学报, 43(2), 234-250.
周志华. (2016). 机器学习. 清华大学出版社.
Smith, J., & Johnson, M. (2020). Deep Learning for Natural Language Processing. In Proceedings of ACL 2020 (pp. 1234-1245).
```

**2. in_text_citations.json**（文内引用映射）：
```json
{
  "张三2020": "(张三, 李四, & 王五, 2020)",
  "周志华2016": "(周志华, 2016)",
  "Smith2020": "(Smith & Johnson, 2020)"
}
```

## 常见问题

### 问题1：如何处理无作者的文献？

**APA**：使用标题代替作者
```
(论文标题, 2020)
```

**MLA**：使用标题前几个词
```
("论文标题" 45)
```

**GB/T 7714**：使用"佚名"
```
佚名. 论文标题[J]. 期刊名称, 2020, 35(2): 45-67.
```

### 问题2：如何引用同一作者的多篇文献？

**按年份排序**：
```
(张三, 2018, 2020)
```

**同年份文献**：
```
(张三, 2020a, 2020b)
```

### 问题3：如何引用外文文献？

- 中文论文引用外文文献，一般保留原文
- 如需翻译，在原文后括号内注明中文译名
- 作者姓名保持原文格式
