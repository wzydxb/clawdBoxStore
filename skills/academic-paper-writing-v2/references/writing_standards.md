## 期刊论文规范

### 结构要求

**标准结构（IMRAD）**：
- **Abstract（摘要）**：150-300字，包含研究背景、方法、结果、结论
- **Introduction（引言）**：研究背景、文献综述、研究问题、研究意义
- **Methods（方法）**：研究设计、数据来源、分析方法
- **Results（结果）**：研究发现、数据呈现
- **Discussion（讨论）**：结果解释、理论贡献、局限性
- **Conclusion（结论）**：核心发现总结、未来研究方向
- **References（参考文献）**：按期刊要求格式排列

### 字数要求

| 期刊类型 | 建议字数 |
|---------|---------|
| 快报/通讯 | 2000-3000字 |
| 研究论文 | 5000-8000字 |
| 综述论文 | 8000-15000字 |

### 高影响因子期刊特点

**Nature/Science/Cell**：
- 强调研究的重大创新性和广泛影响力
- 摘要采用"导语式"写法
- 图表质量要求极高
- 补充材料详细完整

**专业顶级期刊**：
- 注重方法论的严谨性
- 强调学科贡献
- 文献综述需全面

## 学位论文规范

### 结构要求

**博士学位论文**：
```
封面
摘要（中英文）
目录
第1章 绪论
  1.1 研究背景
  1.2 研究意义
  1.3 国内外研究现状
  1.4 研究内容与结构安排
第2章 理论基础与文献综述
第3章 研究设计与方法
第4章 研究结果与分析
第5章 讨论
第6章 结论与展望
参考文献
附录
致谢
攻读学位期间发表的学术论文
```

**硕士学位论文**：
- 结构同博士论文，篇幅控制在3-5万字
- 创新性要求低于博士论文

### 字数要求

| 学位类型 | 字数要求 |
|---------|---------|
| 博士论文 | 8-15万字 |
| 硕士论文 | 3-5万字 |
| 学士论文 | 1-2万字 |

### 特殊要求

1. **创新性声明**：需声明论文的原创性和创新点
2. **文献综述章节**：必须独立成章，全面梳理研究现状
3. **方法论章节**：详细说明研究设计，确保可重复性
4. **附录完整**：问卷、访谈提纲、代码等需放入附录

## 会议论文规范

### 结构要求

**计算机科学会议（CVPR/ICCV/NeurIPS等）**：
- 双栏格式
- 页数限制严格（通常6-8页）
- 结构：Abstract → Introduction → Related Work → Method → Experiments → Conclusion
- 参考文献不计入页数

**工程类会议**：
- 格式由会议组委会规定
- 通常包含全文和扩展版本

### 字数与页数要求

| 会议类型 | 页数限制 |
|---------|---------|
| 短文 | 4-6页 |
| 长文 | 8-12页 |
| 海报 | 2-4页 |

### 注意事项

1. **篇幅控制**：严格遵守页数限制，超出将直接拒稿
2. **格式统一**：使用会议官方模板
3. **图表紧凑**：合理布局，确保在限制页数内完整表达
4. **补充材料**：重要但篇幅受限的内容可放入补充材料

## 通用格式要求

### 标题层级

```
一级标题：第X章 标题（三号黑体，居中）
二级标题：X.X 标题（四号黑体，左对齐）
三级标题：X.X.X 标题（小四号黑体，左对齐）
四级标题：X.X.X.X 标题（五号宋体，左对齐）
```

### 图表规范

**图片要求**：
- 分辨率≥300dpi
- 格式优先使用矢量图（PDF/EPS/SVG）
- 编号格式：图X-X
- 标题置于图下方

**表格要求**：
- 使用三线表
- 编号格式：表X-X
- 标题置于表上方

### 公式规范

- 公式居中排列
- 编号右对齐：(X-X)
- 变量使用斜体，算符使用正体

### 页面设置

| 项目 | 要求 |
|-----|------|
| 纸张 | A4 |
| 页边距 | 上2.5cm，下2.5cm，左3cm，右2.5cm |
| 行距 | 1.5倍行距或固定值20磅 |
| 字体 | 中文：宋体；英文：Times New Roman |
| 字号 | 正文：小四号；标题：见层级规定 |

### 参考文献

- 引用格式：参见 [citation_formats.md](citation_formats.md)
- 引用数量：期刊论文30-50篇，学位论文100篇以上
- 近5年文献占比：≥50%

## 常见问题与解决方案

### 问题1：文献综述如何避免简单罗列？

**解决方案**：
- 按主题或方法论分类综述
- 指出现有研究的不足和争议点
- 明确本研究的位置和贡献

### 问题2：讨论部分如何深入？

**解决方案**：
- 解释结果的理论意义
- 与已有研究对比分析
- 讨论研究的局限性和适用范围
- 提出未来研究方向

### 问题3：如何提高论文的可读性？

**解决方案**：
- 使用清晰的逻辑连接词
- 段落不宜过长（建议5-8句）
- 关键信息使用图表呈现
- 避免过度专业化的术语堆砌
