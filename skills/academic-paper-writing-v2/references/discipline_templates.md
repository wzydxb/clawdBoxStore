## 理工科论文结构

### 计算机科学

**标准结构（会议论文）**：
```
Abstract
1. Introduction
   - Motivation
   - Challenges
   - Contributions
2. Related Work
3. Proposed Method
   - Problem Formulation
   - Method Details
   - Algorithm
4. Experiments
   - Datasets
   - Implementation Details
   - Results and Analysis
   - Ablation Study
5. Conclusion
References
```

**关键特点**：
- 强调创新点和贡献
- 方法论需要数学公式和算法伪代码
- 实验部分需要对比实验和消融实验
- 图表需展示性能对比

**常用引用格式**：IEEE、ACM

### 电子工程

**标准结构**：
```
Abstract
1. Introduction
2. System Model / Problem Formulation
3. Proposed Method
   - Theoretical Analysis
   - Algorithm Design
4. Simulation Results
   - Simulation Setup
   - Performance Evaluation
5. Conclusion
References
```

**关键特点**：
- 理论分析部分需严谨的数学推导
- 实验称为"仿真"（Simulation）
- 需要理论分析与仿真结果对照

### 机械工程

**标准结构**：
```
Abstract
1. Introduction
2. Literature Review
3. Methodology
   - Design
   - Analysis
   - Experiment Setup
4. Results and Discussion
5. Conclusions
References
```

**关键特点**：
- 强调实验验证
- 需要详细的实验装置描述
- 结果包含理论值与实验值对比

## 人文社科论文结构

### 文学研究

**标准结构**：
```
摘要
1. 引言
   - 研究对象与问题
   - 研究意义
2. 文献综述
3. 理论框架
4. 文本分析
   - 分析维度1
   - 分析维度2
   - 分析维度3
5. 结论
参考文献
```

**关键特点**：
- 强调理论框架
- 分析采用文本细读
- 注重论证的逻辑性
- 引用经典文献

**常用引用格式**：MLA、Chicago

### 历史学研究

**标准结构**：
```
摘要
1. 引言
   - 研究问题
   - 史料说明
2. 历史背景
3. 核心论述
   - 论点1
   - 论点2
   - 论点3
4. 结论
参考文献
```

**关键特点**：
- 强调史料运用
- 论证需史论结合
- 注重历史脉络梳理

**常用引用格式**：Chicago（注释-参考文献制）

### 社会学研究

**标准结构**：
```
摘要
1. 引言
   - 研究背景与问题
   - 研究意义
2. 文献综述
   - 理论视角
   - 经验研究
3. 研究方法
   - 研究设计
   - 数据来源
   - 分析方法
4. 研究发现
5. 讨论与结论
参考文献
```

**关键特点**：
- 理论框架需明确
- 方法论部分需详细
- 数据分析需严谨

**常用引用格式**：APA

### 教育学研究

**标准结构**：
```
摘要
1. 问题提出
   - 研究背景
   - 研究问题
2. 文献综述
3. 研究设计
   - 研究对象
   - 研究方法
   - 研究工具
4. 研究结果
5. 讨论与建议
6. 结论
参考文献
```

**关键特点**：
- 强调实践意义
- 需要提出教育建议
- 注重研究伦理说明

## 医学论文结构

### 临床研究

**标准结构（IMRAD）**：
```
Abstract
   - Background
   - Methods
   - Results
   - Conclusions
1. Introduction
2. Methods
   - Study Design
   - Participants
   - Interventions
   - Outcomes
   - Statistical Analysis
3. Results
   - Participant Flow
   - Baseline Data
   - Outcomes
4. Discussion
   - Principal Findings
   - Comparison with Other Studies
   - Limitations
5. Conclusions
References
```

**关键特点**：
- 必须包含伦理声明
- 需要临床试验注册号
- 方法学需遵循CONSORT声明
- 统计方法需详细描述

**常用引用格式**：AMA、Vancouver

### 基础医学研究

**标准结构**：
```
Abstract
1. Introduction
2. Materials and Methods
   - Cell Culture / Animal Models
   - Experimental Procedures
   - Statistical Analysis
3. Results
4. Discussion
5. Conclusions
References
```

**关键特点**：
- 实验方法需详细可重复
- 需要动物伦理或细胞株说明
- 结果以图表为主

## 学科适配要点

### 写作风格差异

| 学科类型 | 语言风格 | 论证方式 | 文献引用 |
|---------|---------|---------|---------|
| 理工科 | 精确简洁 | 数据+公式 | 最新文献为主 |
| 人文社科 | 论述性 | 文本+逻辑 | 经典文献重要 |
| 医学 | 规范严谨 | 循证医学 | 近期文献为主 |
| 商科 | 平衡 | 理论+实证 | 理论文献重要 |

### 图表使用差异

| 学科类型 | 图表类型 | 使用频率 |
|---------|---------|---------|
| 理工科 | 流程图、算法框图、性能曲线 | 高 |
| 人文社科 | 引文表、概念图 | 中 |
| 医学 | 流程图、生存曲线、森林图 | 高 |
| 商科 | 统计表、概念模型图 | 高 |

### 参考文献数量建议

| 论文类型 | 理工科 | 人文社科 | 医学 | 商科 |
|---------|--------|---------|------|------|
| 期刊论文 | 30-50篇 | 40-60篇 | 30-50篇 | 40-60篇 |
| 学位论文 | 100-150篇 | 150-200篇 | 100-150篇 | 120-180篇 |

### 关键术语使用

**理工科**：
- 使用数学公式表达
- 算法、模型、优化、性能

**人文社科**：
- 使用理论概念
- 语境、话语、范式、建构

**医学**：
- 循证医学术语
- 随机对照、队列研究、Meta分析

**商科**：
- 管理学术语
- 竞争优势、战略定位、价值链

## 模板选择指南

### 根据论文类型选择

| 论文类型 | 推荐模板 | 特殊要求 |
|---------|---------|---------|
| SCI期刊论文 | 理工科模板 | 英文写作，图表精美 |
| CSSCI期刊论文 | 人文社科模板 | 注重理论对话 |
| 北大核心期刊 | 按学科选择 | 强调学术规范 |
| 博士学位论文 | 学科对应模板 | 文献综述独立成章 |
| 硕士学位论文 | 学科对应模板 | 创新性要求较低 |
| 国际会议论文 | 理工科模板 | 页数限制严格 |

### 根据研究方法选择

| 研究方法 | 适用结构 | 特殊章节 |
|---------|---------|---------|
| 定量研究 | 实证型结构 | 数据、变量、模型 |
| 定性研究 | 论述型结构 | 案例描述、文本分析 |
| 混合研究 | 综合结构 | 整合定量与定性部分 |
| 文献研究 | 综述型结构 | 文献分类、比较分析 |
| 实验研究 | IMRAD结构 | 实验设计、被试信息 |
