# 学术论文撰写技能

## 任务目标

- **全流程支持**：从目标确认到定稿交付的五步标准化流程，每步设有门禁机制确保质量
- **多学科适配**：理工科（IMRAD结构）、人文社科（论述型）、医学（临床/基础）、商科（实证型）四大学科模板
- **引用规范化**：支持 APA/MLA/Chicago/GB-T 7714 四种格式，自动生成参考文献列表和文内引用
- **文档输出**：生成符合期刊/学校格式要求的 Word（.docx）文档，含封面、摘要、正文、参考文献完整结构
- **质量评估**：四维评分体系（结构25分+内容30分+规范25分+语言20分），≥80分方可交付

**触发条件**：
- "帮我写一篇关于X的学术论文/综述"
- "我需要撰写学位论文/期刊投稿/会议论文"
- "我已有文献检索结果，需要整合撰写论文"
- "帮我规划第X章的结构和内容"

## 五步标准化流程

### 步骤1：明确写作目标与规范要求

**执行内容**：
1. 解析用户输入，识别：
   - 研究主题与研究方向
   - 学科领域（理工科/人文社科/医学等）
   - 论文类型（期刊论文/学位论文/会议论文）
   - 目标期刊/会议（如有）

2. 明确规范要求：
   - 字数要求与篇幅限制
   - 章节结构规范
   - 引用格式标准（APA/MLA/Chicago/GB/T 7714）
   - 特殊格式要求（图表、公式、附录等）

3. 参考规范文档：
   - 阅读 [references/writing_standards.md](references/writing_standards.md) 了解各类论文写作规范
   - 阅读 [references/discipline_templates.md](references/discipline_templates.md) 了解学科适配要点

**输出物**：《写作规范确认表》

**门禁机制**：必须完成《写作规范确认表》并经用户确认后，才能进入步骤2

---

### 步骤2：文献调研与素材整理

**执行内容**：

1. **接收文献检索结果**：
   - 支持从学术论文检索技能导入JSON格式结果
   - 支持用户直接提供文献列表

2. **文献质量分析**：
   - 主题相关性评估
   - 时效性筛选（优先近5年文献）
   - 权威性判断（期刊级别、引用次数）

3. **素材提取与整理**：
   - 提取关键观点、方法论、数据支撑
   - 构建文献矩阵（作者、年份、核心观点、引用价值）
   - 识别研究空白与创新点

4. **格式化参考文献**：
   - 调用脚本：`scripts/citation_formatter.py --format <格式> --input <文献数据> --output references.json`
   - 生成标准格式的参考文献列表

**输出物**：《文献调研报告》+《参考文献列表》

**门禁机制**：必须完成文献调研报告（包含文献矩阵和参考文献列表）后，才能进入步骤3

---

### 步骤3：论文结构规划与大纲生成

**执行内容**：

1. **选择论文结构模板**：
   - 根据论文类型和学科领域，参考 [references/discipline_templates.md](references/discipline_templates.md)
   - 确定章节结构框架

2. **生成章节大纲**：
   - 基于文献调研结果，规划各章节核心内容
   - 明确各章节的核心论点与论据支撑
   - 设计论证逻辑链条

3. **大纲结构示例**（期刊论文）：
   ```
   1. 摘要（200-300字）
   2. 引言
      2.1 研究背景
      2.2 文献综述
      2.3 研究问题与目标
   3. 研究方法
      3.1 研究设计
      3.2 数据来源
      3.3 分析方法
   4. 研究结果
   5. 讨论
   6. 结论
   7. 参考文献
   ```

**输出物**：《论文大纲》

**门禁机制**：必须经用户确认《论文大纲》后，才能进入步骤4

---

### 步骤4：内容撰写与文献引用

**执行内容**：

1. **分章节撰写**（智能体主导）：
   - 按大纲逐章节生成内容
   - 确保内容逻辑连贯、论证充分
   - 体现学术语言风格

2. **文献引用融合**：
   - 在论证关键点插入文献引用
   - 确保引用位置恰当、数量合理
   - 使用标准引用格式（见 [references/citation_formats.md](references/citation_formats.md)）

3. **格式规范化处理**：
   - 调用脚本：`scripts/paper_template_filler.py --template <模板> --data <论文数据> --output paper.docx`
   - 生成符合格式要求的文档

**输出物**：《论文初稿》

**门禁机制**：完成初稿并通过格式校验后，才能进入步骤5

---

### 步骤5：质量评估与润色优化

**执行内容**：

1. **逻辑连贯性检查**：
   - 论证链条完整性
   - 章节衔接流畅性
   - 结论与研究问题对应性

2. **学术规范性检查**：
   - 引用格式合规性
   - 术语使用准确性
   - 图表标注规范性

3. **语言质量优化**：
   - 语法与表达修正
   - 学术风格润色
   - 冗余内容精简

4. **质量评估**：
   - 参考 [references/quality_criteria.md](references/quality_criteria.md)
   - 从结构、内容、格式、语言四个维度评分
   - 生成《质量评估报告》

**输出物**：《质量评估报告》+《论文终稿》

**门禁机制**：质量评分达标（≥80分）后交付最终稿，否则返回相应步骤优化

## 注意事项

1. **流程严谨性**：严格遵循五步流程和门禁机制，不得跳跃执行
2. **文献质量**：优先使用近5年、高影响因子期刊文献
3. **引用规范**：引用格式必须符合目标期刊要求
4. **原创性保障**：内容生成基于文献综合分析，避免直接复制
5. **学科适配**：不同学科有特定的写作规范，注意参考discipline_templates.md

## 使用示例

### 示例1：期刊论文撰写（完整流程）

```
用户：研究主题是"AI辅助医疗影像诊断"，目标投 Nature Medicine，
      已有文献检索结果 JSON 文件，需要完整论文

技能执行：
步骤1 → 确认：医学领域，IMRAD结构，AMA引用格式，7000字，英文撰写
步骤2 → 导入45篇文献，筛选A级21篇，格式化参考文献列表（AMA格式）
步骤3 → 输出大纲（Introduction/Methods/Results/Discussion/Conclusion）
         ，用户确认后进入撰写
步骤4 → 逐节生成内容，关键论点插入文献引用，调用脚本生成 paper.docx
步骤5 → 质量评分92/100（优秀），交付论文终稿 + 质量报告
```

### 示例2：学位论文章节指导

```
用户：博士论文（计算机科学），第1-3章已完成，
      需要指导第4章"实验设计与结果分析"

技能执行：
1. 阅读已完成章节，理解研究问题与方法论
2. 规划第4章结构：
   4.1 实验设置（数据集/硬件环境/超参数配置）
   4.2 基线方法与评估指标
   4.3 主实验结果（含消融实验）
   4.4 误差分析与讨论
3. 提供各节撰写要点和注意事项
4. 质量评估，输出修改建议
```

### 示例3：会议论文快速生成

```
用户：投 CVPR 2025，主题"基于扩散模型的图像分割"，8页限制，3天后截止

技能执行：
步骤1 → 确认 CVPR 格式（双栏、8页正文+参考文献、IEEE格式引用）
步骤2 → 快速整理用户提供的实验数据和方法说明
步骤3 → 精简大纲（突出 Contribution 和与 SOTA 对比，压缩 Related Work）
步骤4 → 优先生成 Abstract/Introduction/Method/Experiments，
         调用脚本生成符合 CVPR 模板的 .docx 初稿
步骤5 → 格式校验 + 语言润色，确保在8页内完整表达
```

### 示例4：参考文献格式转换

```
用户：我有一批文献数据（JSON格式），需要转换成 GB/T 7714 格式

技能执行：
直接调用 citation_formatter.py，输出：
- formatted_references.txt（完整参考文献列表）
- in_text_citations.json（文内引用映射，如"（张三等, 2023）"）
支持 APA/MLA/Chicago/GB-T 7714 四种格式一键转换
```
