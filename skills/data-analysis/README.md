# 数据分析与可视化

## 简介

数据分析和可视化工具，支持数据库查询、报告生成、电子表格自动化等场景，内置图表选择指南、分析技巧和常见陷阱提示。

## 主要功能

- 数据库查询：支持 SQL 查询辅助，自动生成查询语句
- 图表生成：根据数据类型推荐合适的图表类型，生成可视化
- 报告生成：将分析结果整理成结构化报告
- 电子表格自动化：处理 CSV/Excel 数据，执行批量计算和转换
- 指标合约管理：定义和追踪关键业务指标（metric contracts）
- 决策简报：将数据分析结果转化为决策支持文档

## 使用方式

直接将数据文件或数据库查询需求发给助手。

参考文件（skill 内置）：
- `chart-selection.md`：图表类型选择指南
- `decision-briefs.md`：决策简报模板
- `metric-contracts.md`：指标定义规范
- `pitfalls.md`：常见分析陷阱
- `techniques.md`：分析技巧手册

示例提示：
- "分析这份销售数据，找出增长趋势"
- "帮我写一个 SQL 查询，统计每月活跃用户数"
- "把这个 Excel 表格里的数据做成可视化报告"
- "这个指标下降了，帮我分析可能的原因"

## 依赖/前置条件

- 无需 API Key
- 处理本地文件需要 Python（pandas、matplotlib）或 Node.js 环境
- 数据库查询需要提供连接信息或直接粘贴数据

## 注意事项

- 上传数据前请确认已脱敏，不要包含真实姓名、手机号、身份证等个人信息
- 大数据集（超过 10 万行）建议先抽样分析，再全量处理
- 图表生成依赖具体运行环境，部分功能需要配合 chart-image skill 使用
