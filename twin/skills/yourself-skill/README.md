# yourself-skill

## 简介

将自己蒸馏成一个 AI Skill，通过导入微信聊天记录、社交媒体内容、个人文档等多平台数据，生成专属数字分身。

## 主要功能

- 解析微信、QQ 聊天记录，提取个人表达习惯与语言风格
- 分析社交媒体内容，提炼性格特征与价值观
- 整合多源数据，构建完整的个人 Persona
- 支持版本管理，持续迭代优化数字分身
- 支持多次导入数据后合并更新

## 使用方式

1. 准备数据：导出微信/QQ 聊天记录、社交媒体内容、个人文档
2. 通过 OpenClaw 调用此 skill，按提示上传数据
3. skill 依次完成：数据解析 → Persona 分析 → 数字分身构建
4. 生成完成后可直接与数字分身对话，或导出为独立 skill

### 工具说明

| 工具 | 用途 |
|------|------|
| `wechat_parser` | 解析微信聊天记录 |
| `qq_parser` | 解析 QQ 聊天记录 |
| `social_parser` | 解析社交媒体内容 |
| `photo_analyzer` | 分析照片元数据与内容 |
| `skill_writer` | 生成最终 skill 文件 |
| `version_manager` | 管理 skill 版本 |

## 依赖 / 前置条件

- 无需额外 API key
- 数据文件需为本地可访问路径

## 注意事项

- 聊天记录等个人数据仅在本地处理，不会上传到外部服务
- 数据量越大，生成的数字分身越准确
- 建议定期更新数据以保持分身的时效性
